## Usage

* install Martini

        go get github.com/go-martini/martini

* install websocket

        go get github.com/gorilla/websocket

* install this ChatRoom

        go get github.com/renleimlj/ChatRoom

* use this ChatRoom

        go run main.go

open your browser and input `http://localhost:3000/`
